# Port-Scanner-Banner-Grabber
Inicializamos en una ip objetivo señalando los puertos disponibles (Port Scanner) y el nombre del banner en cada uno de los puertos disponibles (Banner Grabber)
Es necesario cambiar la 2 linea de codigo de ip_objetivo a la ip que se necesite trabajar
Tambien se puede modificar la 4 linea de codigo de puerto_fin al numero de modulos que se desee analizar
